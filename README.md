# eStore

This is a blockchain based eStore where user can buy books from the store. The project is developed on etherium based blockchain.


## Tech-Stack
1. Truffle
2. Truffle react-box
3. Solidity
4. Javascript
5. Ganche-cli
6. Metamask
7. Remix


## Quick Start

1. Open truffle develop console:

        truffle develop

2. Compile contracts:

        truffle compile

3. Migrate contracts:

        truffle migrate --reset

4. Test contracts:

        truffle test
